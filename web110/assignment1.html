I learned that IP addresses can be be used to find out someone's location through geolocation, although the results are not always accurate. I then proceeded to use whatismyipaddress.com to find out my own IPv4 address and learn the difference between and IPv4 address and a IPv6. A IPv4 address is the standard IP version, and is shorter in length, while a IPv6 address is a newer formatted IP address that is longer in length. My own computer does not use a IPv6, as I found out, since it wasn't detected using the given websites.
